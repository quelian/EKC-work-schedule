## 🎨 Концепция

Современный, чистый интерфейс в стиле Apple iOS/macOS с пастельными тонами и интуитивной навигацией.

### Ключевые принципы:

- ✨ **Минимализм** — много воздуха, ничего лишнего
- 🎨 **Пастельные тона** — приятные градиенты и цвета
- 📱 **Адаптивность** — работает на всех устройствах
- 🎯 **Интуитивность** — понятно с первого взгляда
- ⚡ **Быстродействие** — плавные анимации и переходы

**Файл:** `app/templates/employees_new.html`

- ✏️ Редактирование и деактивация

### 3. Ограничения (`/constraints`)

**Файл:** `app/templates/constraints.html`

**Вкладки:**

1. 📚 **Учёба** — строгие ограничения (нельзя ставить работу)
2. 💭 **Пожелания** — мягкие ограничения (можно в крайнем случае)
3. 📥 **Импорт** — загрузка из документов сотрудников

**Добавление учёбы:**

```
[Выберите сотрудника ▼] [Дата] [14:00] - [18:00] [Добавить]
```

**Добавление пожелания:**

```
[Выберите сотрудника ▼] [Дата] [Хочу выходной ▼] [Сохранить]
```

| Настройки   | `/settings`    | Параметры системы         |

## 🔧 Интеграция в существующее приложение

### Шаг 1: Обновить main.py

Добавить импорт нового шаблона:

```python
from .main_new import home, employees_page, constraints_page
```

### Шаг 2: Добавить маршруты

```python
app.add_route("/", home)
app.add_route("/employees", employees_page)
app.add_route("/constraints", constraints_page)
```

### Шаг 3: Обновить навигацию

В `base_new.html` уже есть готовая навигация с боковой панелью.

## 🎯 Лучшие практики

### 1. Иконки

Используйте emoji для наглядности:

- 👥 Сотрудники
- 📅 Календарь
- 📊 График
- ⚙️ Настройки
- ✅ Успех
- ❌ Ошибка
- ⚠️ Предупреждение

### 2. Карточки

Каждая карточка должна:

- Иметь тень (`box-shadow`)
- Закруглённые углы (`border-radius: 16px`)
- Плавный hover-эффект
- Белый фон

### 3. Кнопки

- **Основные**: Градиент синий → фиолетовый
- **Вторичные**: Серый фон + синий текст
- **Опасные**: Красный фон

### 4. Формы

- Поля с закруглёнными углами
- Голубая обводка при фокусе
- Placeholder с примером ввода
