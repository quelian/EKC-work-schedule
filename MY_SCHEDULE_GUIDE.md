# 👤 Личный кабинет сотрудника

## 🎯 Назначение

Личная страница каждого сотрудника для **самостоятельного** управления своими пожеланиями и ограничениями.

**Цель:** Избавиться от анализа Word-файлов — пусть сотрудники сами заполняют свои данные!

---

## 🚀 Как использовать

### Для сотрудника:

1. **Открой личную страницу:**
   ```
   http://127.0.0.1:8001/my-schedule
   ```

2. **Выбери себя из списка**

3. **Заполни когда не можешь работать:**
   - Нажми "⛔ Когда не могу работать"
   - Выбери дату
   - Укажи время (например, 14:00 - 16:00)
   - Нажми "Добавить"

4. **Добавь пожелания:**
   - Нажми "💭 Пожелания"
   - Выбери дату
   - Выбери: "Хочу выходной" или "Хочу работать"
   - Нажми "Сохранить"

5. **Просмотри свой график:**
   - Нажми "📊 Мой график"
   - Увидишь все свои смены

---

## 📱 Интерфейс

### Вкладки:

1. **⛔ Когда не могу работать** (строгие ограничения)
   - Учеба, пары, экзамены
   - Врачи, встречи
   - Другие дела
   
   ❌ **Нельзя ставить работу в это время!**

2. **💭 Пожелания** (мягкие ограничения)
   - "Хочу выходной" — желательно не ставить
   - "Хочу работать" — приоритет при назначении
   
   ⚠️ **Можно нарушить в крайнем случае**

3. **📊 Мой график**
   - Все назначенные смены
   - Даты и время
   - Длительность

## 📋 Примеры использования

### Пример 1: Студент указывает пары

**Ситуация:** Настя учится по понедельникам и средам с 14:00 до 16:00

**Действия:**
1. Открывает `/my-schedule?employee_name=Настя`
2. Переходит в "⛔ Когда не могу работать"
3. Добавляет:
   - Понедельник 14:00-16:00
   - Среда 14:00-16:00
4. Повторяет для каждой недели

**Результат:** Настя не будет назначена на это время!

---

### Пример 2: Пожелание выходного

**Ситуация:** У Юли день рождения 15 марта

**Действия:**
1. Открывает личную страницу
2. Переходит в "💭 Пожелания"
3. Добавляет:
   - Дата: 15 марта
   - Тип: "Хочу выходной"
   - Комментарий: "День рождения"

**Результат:** Юля скорее всего получит выходной (если есть кем заменить)

---

### Пример 3: Хочу работать больше

**Ситуация:** Артём хочет работать в определённые дни

**Действия:**
1. Открывает личную страницу
2. Переходит в "💭 Пожелания"
3. Добавляет несколько дней с типом "Хочу работать"

**Результат:** Артём получит приоритет на эти дни

## 🔄 Интеграция с планировщиком

При построении графика система **автоматически** учитывает:

1. **Unavailable (строгие):**
   ```python
   if employee has unavailable at this time:
       skip  # Нельзя ставить!
   ```

2. **Preferences (мягкие):**
   ```python
   if employee prefers_off:
       score -= 50  # Штраф, но можно
   
   if employee prefers_work:
       score += 50  # Бонус
   ```
