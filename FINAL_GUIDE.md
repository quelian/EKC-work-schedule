## 🔧 Интеграция со старым приложением:

Если хочешь использовать новый дизайн в старом main.py:

```python
# В app/main.py добавь:
from app.main_new import (
    home, employees_page, constraints_page,
    calendar_page, schedule_page, schedule_editor_page,
    settings_page, my_schedule_page,
    add_unavailable, delete_unavailable,
    add_preference, delete_preference
)

app.add_route("/new", home)
app.add_route("/new/employees", employees_page)
app.add_route("/new/constraints", constraints_page)
app.add_route("/new/calendar", calendar_page)
app.add_route("/new/schedule", schedule_page)
app.add_route("/new/schedule/editor", schedule_editor_page)
app.add_route("/new/settings", settings_page)
app.add_route("/new/my-schedule", my_schedule_page)

# POST маршруты
app.add_route("/new/my-schedule/unavailable/add", add_unavailable, methods=["POST"])
app.add_route("/new/my-schedule/unavailable/delete", delete_unavailable, methods=["POST"])
app.add_route("/new/my-schedule/preferences/add", add_preference, methods=["POST"])
app.add_route("/new/my-schedule/preferences/delete", delete_preference, methods=["POST"])
```
