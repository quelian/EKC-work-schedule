# 📊 Редактор графика

## 🎯 Возможности

**Мощный. Интуитивный. Красивый.**

### ✏️ Редактирование

1. **Режим редактирования**
   - Включи кнопкой "✏️ Режим редактирования"
   - Кликни на любую дату чтобы добавить смену
   - Кликни на смену чтобы редактировать

2. **Inline редактирование**
   - Клик → Редактируй → Сохрани
   - Мгновенное применение
   - Визуальная обратная связь

3. **Быстрые действия**
   - 🔄 Поменять местами
   - 📋 Копировать смену
   - 🗑️ Удалить смену

## 🔧 Функции редактирования

### 1. Изменение времени
```
[08:00] - [14:00] → [09:00] - [17:00]
```

### 2. Замена сотрудника
```
Иванов → Петров (выбор из списка)
```

### 3. Копирование смены
```
Выбрать смену → Копировать → Выбрать дату → Вставить
```

### 4. Обмен сменами
```
Смена A ↔ Смена B
(Даты или сотрудники меняются местами)
```

## 🎯 Примеры использования

### Пример 1: Быстрая замена

**Ситуация:** Иванов заболел, нужно заменить на Петрова

**Действия:**
1. Включить режим редактирования (`E`)
2. Кликнуть на смену Иванова
3. Выбрать Петрова из списка
4. Сохранить (`S`)

**Время:** 10 секунд ⚡

---

### Пример 2: Массовое копирование

**Ситуация:** Нужно поставить одинаковые смены на неделю

**Действия:**
1. Создать смену в понедельник
2. Копировать (`C`)
3. Кликнуть на вторник → Вставить
4. Повторить для остальных дней

**Время:** 1 минута ⚡

---

### Пример 3: Обмен выходными

**Ситуация:** Иванов и Петров хотят поменяться выходными

**Действия:**
1. Включить режим редактирования
2. Кликнуть на смену Иванова
3. Нажать "🔄 Поменять местами"
4. Кликнуть на смену Петрова
5. Подтвердить обмен

**Время:** 15 секунд ⚡
