# 🚀 Быстрый старт - Доступ из сети

## 1️⃣ Запусти сервер

```bash
cd "/Users/egor/график работы екц"
./run_server.sh
```

Или вручную:

```bash
source .venv/bin/activate
uvicorn app.main_new:app --host 0.0.0.0 --port 8001 --reload
```

---

## 2️⃣ Узнай свой IP

```bash
ipconfig getifaddr en0
```

**Запомни этот IP!** (например: `192.168.1.100`)

---

## 3️⃣ Открой с любого устройства

### На этом Mac:

```
http://127.0.0.1:8001
```

### С телефона/планшета:

```
http://192.168.1.100:8001
```

*(подставь свой IP)*

### С другого компьютера:

## 🔧 Если не работает

1. **Проверь что сервер запущен:**
  ```bash
   ps aux | grep uvicorn
  ```
2. **Проверь IP:**
  ```bash
   ipconfig getifaddr en0
  ```
3. **Отключи брандмауэр (на время теста):**
  -  → Системные настройки → Брандмауэр
  - Выключить
4. **Перезапусти сервер:**
  ```bash
   ./run_server.sh
  ```
